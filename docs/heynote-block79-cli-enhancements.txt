## Block 79: CLI Enhancements (pi-inspired)

5 features for clawser-cli.js inspired by pi.dev. Full plan: docs/cli-enhancement-plan.md

### Features

1. **Tree-Based Session Branching** (Priority 1, High complexity, 3-4 weeks)
   Git-like branch/checkout/merge for conversations. Adds parentId + branchId to EventLog events,
   branches.json metadata, ASCII tree visualization. New subcommands: branch, checkout, merge, tree, diff.
   Files: clawser-agent.js, clawser-cli.js, clawser-state.js

2. **RPC Mode** (Priority 2, High complexity, 3-4 weeks)
   JSON-RPC 2.0 over stdin/stdout, Unix socket, or HTTP. Methods: sendMessage, listTools, executeTool,
   getStatus, getHistory, setConfig. Bearer-token auth for HTTP. Streaming notifications for response chunks.
   Files: NEW clawser-rpc.js, clawser-cli.js, clawser-agent.js

3. **JSON Output Mode** (Priority 2, Low complexity, 1 week)
   --json flag on all subcommands. Structured envelope: { ok, command, data }. JSONL streaming for prompts.
   Pipe-friendly: clawser history --json | jq '.events[]'. No breaking changes to default output.
   Files: clawser-cli.js, clawser-wsh-cli.js

4. **Hot-Reloading Extensions** (Priority 3, Medium complexity, 2 weeks)
   fs.watch / File System Observer on skill dirs. Re-parse SKILL.md + re-register tools on save.
   Fallback to previous version on parse error. --dev flag auto-enables. Status line notifications.
   Files: NEW clawser-hot-reload.js, clawser-skills.js, clawser-cli.js

5. **Session Sharing** (Priority 3, Medium complexity, 2 weeks)
   Export as standalone HTML viewer, markdown, or JSON. Privacy sanitizer strips API keys/tokens.
   Import from file, URL, or GitHub Gist. Read-only imported sessions forkable for continuation.
   Files: NEW clawser-session-export.js, clawser-cli.js, clawser-state.js

### Implementation Order

Phase 1: JSON Output (1wk, unblocks automation) + Tree Branching (3-4wk, core data model)
Phase 2: RPC Mode (3-4wk, depends on JSON output) + Hot-Reload (2wk, independent)
Phase 3: Session Sharing (2wk, depends on branching for --branch flag)

Total: ~11-13 weeks sequential, ~6-7 weeks with parallelism.
