## Prerequisites

- A modern browser (Chrome 131+ for Chrome AI, any modern browser otherwise)
- Clawser served locally or deployed (see [Quick Start](../../README.md#quick-start))
- At least one LLM provider API key (OpenAI, Anthropic, Groq, etc.) for tutorials beyond 01
