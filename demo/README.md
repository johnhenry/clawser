# Demo Skills

Historical reference skill definitions from the original Clawser prototype. Each subdirectory contains a SKILL.md and supporting files for a specific agent skill.

These demos were built against the Rust/WASM core and may need updates to work with the current pure-JS runtime.

## Skills

| Directory | Description |
|-----------|-------------|
| `code_refactoring` | Automated code refactoring assistance |
| `digital_maintenance` | System cleanup and maintenance tasks |
| `health_investigation` | Health data analysis workflows |
| `research_lab` | Research and investigation assistance |
| `security_sentinel` | Security monitoring and alerting |
| `skill_learning` | Self-directed learning workflows |
| `trip_planning` | Travel planning and itinerary generation |
| `writing_companion` | Writing assistance and editing |
